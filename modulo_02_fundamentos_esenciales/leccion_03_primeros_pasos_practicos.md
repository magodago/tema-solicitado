# Primeros pasos practicos

## Objetivos
- Familiarizar al estudiante con los conceptos básicos y herramientas necesarias para comenzar a trabajar en un proyecto.
- Guiar al estudiante en la configuración inicial del entorno de desarrollo.

## Contenido
En esta lección, se introducirá el ambiente básico necesario para que el estudiante pueda empezar a desarrollar proyectos. Se explicará cómo instalar y configurar el software de desarrollo, así como las herramientas de control de versiones esenciales. Es crucial que los estudiantes comprendan la importancia de mantener un entorno de trabajo organizado y seguro desde el inicio.

Se discutirá también la estructura básica de un proyecto, incluyendo carpetas y archivos clave, y se proporcionarán pautas para organizar el código de manera eficiente. La lección incluirá ejemplos prácticos de cómo crear un repositorio en GitHub o GitLab, y cómo inicializar un nuevo proyecto.

## Ejercicio
El estudiante deberá:
1. Instalar un entorno de desarrollo integrado (IDE) como Visual Studio Code.
2. Configurar Git para su cuenta personal.
3. Crear un nuevo repositorio en GitHub.
4. Clonar el repositorio localmente y crear una estructura básica de proyecto.

## Resumen
- El estudiante aprenderá a configurar el entorno de desarrollo.
- Se enfatizará la importancia de mantener un control de versiones efectivo.
- Los estudiantes practicarán la creación y clonación de repositorios.
